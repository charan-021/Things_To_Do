# Things_To_Do

Hello everyone!

I'm very excited to share that I have successfully developed a website on Oct 7, 2023. It is all about to-do list with user authentication, helps everyone because it's like a friendly guide to your day. It shows you all the things you need to do, making it easier to get stuff done. It stops you from feeling stressed about forgetting things and helps you feel good when you finish tasks. Whether you're a student with schoolwork, a busy worker, or just want a more organized life, a to-do list is like your helpful friend that makes your day smoother.

In my "Things To-do" project, I used frontend technologies such as HTML, CSS, and JavaScript to shape the website's appearance and ensure its smooth functionality. On the backend, I used Django, and I integrated a database in my project for data management and storage. These technologies together enabled me to create a user-friendly and efficient web application.

I worked on this project approximately four weeks, and I did everything by myself, from designing to making it work. I used some tools like Visual Studio Code, github,  vercel and supabase help me along the way. My goal was to make a user-friendly and practical website for managing tasks, and I hope people find it useful.

Website Link: https://things-to-do-git-main-charans-projects.vercel.app/
